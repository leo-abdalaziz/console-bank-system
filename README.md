# Bank System

A banking system written in C++

## Features

## Client's Features

#A client can

-Deposit
-Withdraw
-Check Balance
-transfer

## Employee's Features

#An Employee can

-Add new clients
-Search for existing clients
-List all of the existing clients
-Edit and display any client's info

## Admin's Features

#An admin can

-Do whatever an employee can do
-Add new employees
-Search for existing employees
-Edit and display any employee's info
-List all the existing employees

## Technologies

-C++
-Microsoft Visual Studio
-Git
